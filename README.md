# vue-shop
## 介绍
这是一个基于vue-cli搭建的商城后台的管理系统，主要使用的技术栈有vue.js、axios、element-ui；通过前后端分离，使用axios与服务器进行数据交互，实现用户管理、用户权限设置、商品管理、订单管理等功能。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
